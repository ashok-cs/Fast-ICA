SYNOPSIS


Independent component analysis (ICA) is a recently proposed method as a solution to the blind source separation problem. The objective is to recover the unobserved source signals from the observed mixtures without the knowledge of the mixing coefficients. It has the potential for a wide range of applications in industrial, medical, security, and military fields because it reduces the complex problem of dealing with high-dimensional statistical descriptions to products of one-dimensional density functions[1,2]. For some applications, off-line ICA analysis on a workstation could be adequate, but for a vast majority it is desirable to have a hardware implementation for real-time analysis. In this project we study the ICA algorithm in hardware context and derive the architecture for the real time implementation of ICA algorithm based on low power and high performance reconfigurable computing platforms. Different available reconfigurable platforms will be compared on their power consumption and speed for this purpose.


Appendix A and B give initial simulation results of ICA application in image separation and the flow chart of the ICA algorithm respectively.

Summary:

The project is intended to map the ICA algorithm, a blind source seperation technique, into reconfigurable architectures and compare the performance with respect to power and speed. Some of the challenges involved in the project are understanding of ICA algorithm in hardware context, selection of reconfigurable platforms for the implementation of the algorithm with respect to speed and power consumption, derivation of parallel architecture for the algorithm in selected reconfigurable platforms, mapping of exponential, square root and eigen value decomposition [EVD] functions and estimation of speed and power for the implementation of the algorithm. Platforms that are being consideration for the implementation of the algorithm are Morphosys reconfigurable processor, REMARC reconfigurable processor, PIC microcontroller, intel embedded processor and TI DSP chipset. Other platforms are also being evaluated based on their performance with respect to speed and power consumption before being selected.
